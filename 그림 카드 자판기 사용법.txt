그림 카드 자판기 사용법

<처음 컴퓨터에서 실행할 때>
1. 파이썬(Python) 설치 확인
   - [python-manager-25.2.msix] 파일을 실행해서 설치하거나, Python 공식 홈페이지에서 설치하세요.
   - ★★중요★★: 설치 화면 맨 아래에 있는 [Add Python to PATH] 체크박스를 꼭 체크하고 설치해야 합니다!!

2. 라이브러리 설치
   - [install_requirements.bat] 파일을 더블클릭해서 실행하세요.
   - 검은 창이 뜨고 글자가 쭉 올라가다가 "설치가 완료되었습니다"라고 나오면 성공입니다.
   - 이건 컴퓨터마다 처음 1번만 해주면 됩니다.

<평소에 실행할 때>
1. [index.html] : 그림 자판기 페이지 (아이들이 쓰는 화면)
2. [run_server.bat] : 그림 자판기 관리자 페이지 (선생님이 카드 등록/수정하는 화면)
   - 실행하면 검은 창과 인터넷 창이 같이 뜹니다.
   - 검은 창을 끄면 서버가 종료되니 켜두세요.
