# 구글 스프레드시트 연동 준비 가이드

서버가 구글 시트를 '읽고 쓰기' 위해서는 **디지털 출입증(json 파일)**이 필요합니다.
아래 단계에 따라 키를 발급받아 주세요. (약 5분 소요)

## 1단계: 구글 클라우드 프로젝트 만들기
1.  [구글 클라우드 콘솔](https://console.cloud.google.com/)에 접속합니다.
2.  상단 로고 옆의 **[프로젝트 선택]** 클릭 -> **[새 프로젝트]** 클릭.
3.  프로젝트 이름: `Card-Vending-Machine` (원하는 이름 아무거나)
4.  **[만들기]** 클릭 후 잠시 기다렸다가 해당 프로젝트를 선택합니다.

## 2단계: API 활성화 (Sheets & Drive)
서버가 사용할 기능 2가지를 켜줘야 합니다.
1.  좌측 메뉴: **[API 및 서비스]** -> **[라이브러리]**.
2.  `Google Sheets API` 검색 -> 클릭 -> **[사용]** 버튼 클릭.
3.  다시 라이브러리로 돌아가서 `Google Drive API` 검색 -> 클릭 -> **[사용]** 버튼 클릭.

## 3단계: 서비스 계정(로봇 직원) 만들기
1.  좌측 메뉴: **[API 및 서비스]** -> **[사용자 인증 정보]**.
2.  상단 **[+ 사용자 인증 정보 만들기]** -> **[서비스 계정]** 선택.
3.  이름: `picture-card-bot` (아무거나) -> **[완료]** 클릭.
4.  생성된 이메일 주소(예: `picture-card-bot@.....iam.gserviceaccount.com`)를 **따로 복사해두세요!** (4단계에서 필요)

## 4단계: 키 파일(JSON) 다운로드
1.  방금 만든 서비스 계정(이메일)을 클릭합니다.
2.  상단 탭 중 **[키(Keys)]** 선택.
3.  **[키 추가]** -> **[새 키 만들기]** -> **JSON** 선택 -> **[만들기]**.
4.  파일이 자동으로 다운로드됩니다.
    *   파일명: `project-name-12345.json` 처럼 복잡함.
    *   **중요**: 이 파일 이름을 `credentials.json`으로 바꾸고, **프로젝트 폴더(admin.html 있는 곳)에 넣어주세요.**

## 5단계: 엑셀 파일(구글 시트) 공유하기
마지막으로, 우리 엑셀 파일을 로봇 직원에게 공유해줘야 합니다.
1.  사용하시던 `reference_words.xlsx` 파일을 **구글 드라이브(Google Drive)**에 업로드하고, **구글 스프레드시트**로 엽니다. (또는 새로 만드셔도 됩니다)
2.  우측 상단 **[공유]** 버튼 클릭.
3.  아까 복사해둔 **서비스 계정 이메일** (`...@...iam.gserviceaccount.com`)을 붙여넣고 **[편집자]** 권한으로 공유합니다.
4.  **완료!**

---

### 🚀 준비가 다 되시면...
1.  `credentials.json` 파일을 프로젝트 폴더에 넣으셨나요?
2.  구글 스프레드시트 주소(URL)를 복사해서 저에게 알려주세요. (주소 안에 ID가 들어있어서 필요합니다)

이 두 가지가 준비되면 바로 연동 코드를 작성하겠습니다!
